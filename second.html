Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM (Document Object Model) using JavaScript. Example: getElementById, getElementsByClassName, getElementsByTagName, querySelector, etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets/sets HTML, innerText gets/sets visible text, and textContent gets/sets all text regardless of visibility.

Question-3: What is event delegation in the DOM?

Event delegation is using a single event listener on a parent element to handle events from its children, improving efficiency and handling dynamic elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event on a nested element triggers event handlers on its parent elements as it "bubbles up" the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Create with createElement(), add with appendChild() or insertBefore(), and remove with removeChild() or remove().
